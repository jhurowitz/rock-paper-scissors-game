# rock-paper-scissors-game
A project from The Odin Project that evolves with curriculum progression
- Design a basic game of "rock, paper, scissors" with
only using HTML, JS functions, and console logs
- Add a UI aspect with JS DOM functionality
  - Design visual for rock, paper, and scissors and score tracker
    - Each visual for choice will have an image and a button to select it
    - Score tracker will change based on decision of game
  - Result of each match will be prompted on the screen
- Bonus may be to add gradient, design, and animations (stay tuned)
